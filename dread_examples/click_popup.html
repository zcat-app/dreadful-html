Click Popup

Example:

POP-UP CONTENT

Code:

<div class="click-wrapper">
    <label class="button">
        <input type="checkbox" class="trigger"/>
        CLICK ON ME
    </label>
    <div class="content">
        POP-UP CONTENT
    </div>
</div>

<style>
    .click-wrapper {
        position: relative;
    }
    .click-wrapper > .content {
        display: none;
    }
    .click-wrapper > .button > .trigger {
        display: none;
    }
    .click-wrapper > .button:has(.trigger:checked) ~ .content {
        display: block;
    }
</style>
