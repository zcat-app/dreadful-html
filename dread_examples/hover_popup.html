Hover Popup

Example:

HOVER OVER ME!
POP-UP CONTENT

Code:

<div class="hover-wrapper">
    <div class="button">
        HOVER OVER ME!
    </div>
    <div class="content">
        POP-UP CONTENT
    </div>
</div>

<style>
    .hover-wrapper {
        position: relative;
        display: block;
        float: left;
    }
    .hover-wrapper > .button {
        display: block;
    }
    .hover-wrapper > .content {
        display: none;
        width: 200px;
    }
    .hover-wrapper > .content:hover {
        position: absolute;
        display: block;
    }
    .hover-wrapper > .button:hover ~ .content{
        position: absolute;
        display: block;
    }
</style>
